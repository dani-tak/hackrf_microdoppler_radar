# HackRF Microdoppler Radar

A collection of radar and signal processing experiments using HackRF One SDRs and GNU Radio.

## Projects

- **[Motion Detection](motion_detection/)** — Real-time motion detection using CW Doppler radar at 2.4 GHz with two HackRF Ones
- More coming soon...

## Requirements

- HackRF One SDR(s)
- GNU Radio 3.10+
- gr-osmosdr
- Linux machine or Raspberry Pi
